# 🌍 Infrastructure as Code (IaC)

Este proyecto incluye configuración de infraestructura como código utilizando **Terraform** para aprovisionar recursos en MongoDB Atlas.

## 🧠 ¿Qué incluye?

- Creación de proyecto en MongoDB Atlas
- Creación de cluster (configuración básica)

## 🚀 Uso (opcional)

```bash
terraform init
terraform plan
terraform apply
